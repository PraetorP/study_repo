# BlockChain Example

This project was written to learn the basic concepts of PoW. Based on this [guide](https://blog.logrocket.com/how-to-build-a-blockchain-in-rust/).

<details><summary>CLI Commands</summary>
<p>

#### create block with data

```PowerShell
   create b <data>
```

#### list of blocks

```PowerShell
   ls c
```

#### list of nodes

```PowerShell
   ls p
```

</p>
</details>
